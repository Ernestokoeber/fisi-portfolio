Project Description

Club website for the basketball section of TSV Lindau 1850 e.V., built entirely on a serverless Cloudflare stack. The goal is a modern, maintainable platform with clear information architecture and a robust technical foundation.

The frontend deliberately uses vanilla JavaScript with no framework or npm dependencies, minimizing supply-chain risk. Backend logic runs as a Cloudflare Worker at the edge, data lives in D1 (SQLite), media in R2, and caching in KV. The admin panel is gated by Cloudflare Access as a Zero-Trust proxy.

Technical Approach

  • ° Vanilla JavaScript on the frontend — a deliberate choice against npm dependencies to minimize supply-chain risk
  • ° Cloudflare Workers as the edge runtime for backend logic
  • ° D1 (SQLite) as the database, R2 for object storage, KV for caching
  • ° Cloudflare Pages with CI/CD integration for hosting
  • ° Cloudflare Access as a Zero-Trust proxy for the admin area
  • ° Strict Content Security Policy, completed security audit

Features

  • ° Automated parsing of official DBB game reports (PDF) via a Worker
  • ° Structured storage of box scores in D1, with schedule, league table, and player stats rendered on the frontend
  • ° Admin panel for PDF upload and stat management, secured via Cloudflare Access (Zero-Trust)

Focus Areas

Security: Strict Content Security Policy, Zero-Trust authentication for the admin area, and a completed security audit across the entire architecture.

Architecture: Consistent serverless implementation with a focus on low operating cost and high edge performance. Clear separation between static frontend (Pages), backend logic (Workers), and the data layer (D1 / R2 / KV).
